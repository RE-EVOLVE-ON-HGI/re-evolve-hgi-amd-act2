### Slide 1: Cover Page
*   **Main Title**: RE-EVOLVE ON HGI
*   **Subtitle**: The Human-Governed Adaptive Intelligence Operating System
*   **Theme**: Enterprise AI Agent Control Plane & Containerization
*   **Key visual**: A glowing neural sphere floating over dark space grid coordinates.

### Slide 2: The Core Problem (Fragmented Agents)
*   **Headline**: The Isolation Crisis in Enterprise AI
*   **Bullet Points**:
    *   **No Persistent Memory**: AI agents forget previous contexts, repeating identical tasks.
    *   **Unsafe Execution**: Running un-sandboxed shell scripts exposes backend servers to system commands.
    *   **High Latency Mismatches**: Unpredictable response times block live customer pipelines.

### Slide 3: The Market Need
*   **Headline**: Safe, Private, and High-Throughput Compute
*   **Bullet Points**:
    *   Enterprises cannot upload proprietary data to public shared models.
    *   Need for local model hosting with absolute container isolation.
    *   High-density agent configurations demand massive memory bandwidth (optimized for AMD Instinct MI300X clusters).

### Slide 4: The Convergence Solution
*   **Headline**: Re-Evolve Operating System
*   **Bullet Points**:
    *   **Unified Kernel**: Consolidation of planner systems, sandboxes, and memory vaults.
    *   **Zero-Trust Shielding**: Every execution request is evaluated against safety rule sheets.
    *   **Hardware Optimized**: Pre-configured pipelines for AMD ROCm networks.

### Slide 5: The 7-Layer Blueprint
*   **Headline**: Structural Platform Architecture
*   **Bullet Points**:
    *   *Layer 1 (Ingress)*: Unified CLI, SDK, and frontend interfaces.
    *   *Layer 2 (CENSA Planner)*: Cognitive decompose algorithms generating task graphs (DAGs).
    *   *Layer 3 (Agent Registry)*: Skill mapping registries routing agent selections.
    *   *Layer 4 (Panani X Sandbox)*: Node VM isolated environments.
    *   *Layer 5 (Kavacha Governance)*: Pre-scan safety rules and cost ledgers.
    *   *Layer 6 (Memory Vault)*: PGVector episodic history & Qdrant semantic indexes.
    *   *Layer 7 (AMD Fabric)*: Local Instinct accelerators and ROCm.

### Slide 6: CENSA: Task Decomposer
*   **Headline**: Cognitive Orchestrator
*   **Bullet Points**:
    *   Classifies user objectives into goal categories.
    *   Generates a Directed Acyclic Graph (DAG) detailing execution paths.
    *   Orchestrates parallel step execution among specialized agents.

### Slide 7: Panani X: Absolute Sandboxing
*   **Headline**: Secure Container Runtimes
*   **Bullet Points**:
    *   Executes all untrusted codes inside isolated Node `vm` sandboxes.
    *   Prevents directory traversal, process escaping, and network abuses.
    *   Exposes complete stdout/stderr logs in real-time.

### Slide 8: Memory Vault: Global Context
*   **Headline**: Long-Term Agent Memory
*   **Bullet Points**:
    *   **Episodic Records**: Tracks interaction sequences (stored in PGVector).
    *   **Semantic Matching**: Indexing of documentation files (Qdrant sync).
    *   **Fast Recall**: Allows agents to search previous sessions in under 80ms.

### Slide 9: Kavacha: Zero-Trust Guard
*   **Headline**: Policy Shielding & Auditing
*   **Bullet Points**:
    *   Evaluates all calls against customized compliance rules before execution.
    *   Performs regular expression scans on shell commands to block illegal keywords.
    *   Logs cost metrics, execution duration, and token usage.

### Slide 10: AMD Native Accelerator
*   **Headline**: Built on High-Performance Hardware
*   **Bullet Points**:
    *   **MI300X serving**: Optimizations for vLLM running over local ROCm.
    *   **Sub-500ms Failover**: Automatic routing failovers to Fireworks AI.
    *   **LiteLLM proxy**: Unified API endpoint mapping.
