# Security Policy

## Reporting a Vulnerability
We take the security of Re-Evolve on HGI seriously. If you discover a security vulnerability, please report it privately to the maintainers via GitHub Issues (marked as private) or email.

## Our Process
1. **Triage**: We will acknowledge the report within 48 hours.
2. **Fix**: We will work to resolve the issue in a private branch.
3. **Disclosure**: Once the fix is merged and released, we will publicly acknowledge the contributor and the fix.

## Scope
This policy covers all official releases of the HGI Operating System, including the backend API, frontend dashboard, and SDKs.
