# Contributing to Re-Evolve on HGI

Thank you for your interest in contributing to Re-Evolve on HGI! We welcome community contributions, bug reports, and feature proposals.

## How to Contribute

1. **Check for Issues**: Look through existing issues before submitting a new one.
2. **Fork and Branch**: Create a feature branch off of the `main` branch.
3. **Write Tests**: Ensure any logic changes are accompanied by corresponding test suites.
4. **Follow Style Guidelines**: We use TypeScript and adhere to ESLint configurations defined in the codebase.
5. **Submit a Pull Request**: Provide a detailed description of the problem solved or the feature added.

## Code of Conduct

Please be respectful and collaborative in all communication and interactions.
